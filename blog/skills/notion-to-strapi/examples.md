# 使用範例

## 基本使用

**輸入：**
```
請幫我將這個 Notion 文章轉換為 Strapi 文章：
https://www.notion.so/penguin-cho/你的文章標題-xxx
```

**結果：**
```
✅ 文章建立成功!
📝 文章 ID: 281
🌐 網站已重新生成
```

**自動完成的操作：**
- 從 Notion 獲取文章內容
- 下載所有圖片到本地
- SEO 最佳化標題和描述
- 建立 Strapi 文章
- 觸發網站重新生成

## 自訂格式

**輸入：**
```
請幫我將這個 Notion 文章轉換為 Strapi 文章，並使用自訂的副標題：
https://www.notion.so/penguin-cho/文章標題-xxx

副標題：這是我自訂的副標題內容
```

**說明：** 如果提供自訂副標題，技能會使用您指定的內容而非自動生成。

## 批次處理

**輸入：**
```
請幫我將以下 Notion 文章都轉換為 Strapi 文章：
1. https://www.notion.so/penguin-cho/文章1-xxx
2. https://www.notion.so/penguin-cho/文章2-yyy
3. https://www.notion.so/penguin-cho/文章3-zzz
```

**說明：** 技能會逐一處理每篇文章，最後統一觸發網站重新生成。

## 內容優化

**輸入：** 簡短的 Notion 筆記

**輸出：** 技能會自動優化內容結構：
- 加入引言和結論段落
- 完善文章邏輯和說明
- 優化標題層級和格式
- 確保文章長度和品質符合部落格標準

## 常見問題

### 連線錯誤
**問題：** Notion MCP 連線失敗
**解決：** 檢查 MCP 設定和認證狀態

### API 錯誤
**問題：** Strapi API 認證失敗
**解決：** 檢查 `.env` 檔案中的環境變數設定

### 格式錯誤
**問題：** 生成的 JSON 格式不正確
**解決：** 檢查 BLOG_PRD 規格書，確保所有必要欄位都存在
